# **App Name**: Salud Conectada RD Web

## Core Features:

- Single Page Navigation: SPA structure with smooth navigation between Inicio, Citas, Voluntarios, and Donar sections.
- Interactive Forms: Interactive forms for citas, voluntarios, and donaciones with client-side validation to ensure correct data format.
- Toast Notifications: Display non-intrusive toast notifications upon successful form submissions for user feedback.
- N8N Webhook Integration: Utilize asynchronous JavaScript calls using `fetch` to POST data to N8N webhooks, and handle successes and failures.
- Phone Number Validation: Automatically format and validate phone number entries using Javascript.
- AI-Powered Urgency Detection: A tool that analyzes text inputs from users (such as the Motivo de la Cita) to flag the most urgent cases, or direct users to online resources that can help before a human becomes involved.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke a sense of trust and health.
- Background color: Very light blue (#F0F8FF) for a clean and professional feel.
- Accent color: Soft green (#90EE90) for calls to action and highlights, complementing the blue tones.
- Body and headline font: 'PT Sans', a humanist sans-serif combining modernity and warmth.
- Simple, intuitive icons for navigation and key features.
- Mobile-first, responsive design ensuring a seamless experience across devices.
- Subtle animations for form submission confirmations and page transitions.
